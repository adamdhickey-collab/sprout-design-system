# Brand photography — placeholder set

These 12 images were extracted directly from `Brand Guidelines March 2026.pdf`
(the brand's own Photography chapter, pages 80–92) as a curated, on-brand
placeholder set for the leaf-window / leaf-over-image components, since a
licensed photography library wasn't available at the time these components
were built.

**Resolution is screen-only (~590×440, 72dpi)** — the source PDF doesn't embed
higher-resolution originals. Fine for cards and demos; not for print or
large hero use.

**Swap these for the real, licensed Cargill photography library** once
available. Filenames describe subject/category (`world-`, `systems-`,
`portrait-`, `people-`) loosely matching the brand book's own categories
(Our people / Our systems / Our world) so replacements are easy to map.
