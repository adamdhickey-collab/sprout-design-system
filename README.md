# Sprout Design System

A working design-system styleguide for Cargill Sprout — foundations, system-level tokens, and 48 components, split across a few pages so each stays a reasonable size:

| Page | Contents |
|---|---|
| `index.html` | Home + Foundations: Color, Typography, Layout & shape, Iconography, Logo |
| `system.html` | Themes, Elevation, Brand expression, Illustration & diagrams, Data visualization |
| `forms.html` / `content.html` / `navigation.html` | Components, grouped by Forms & inputs / Layout & content / Navigation & feedback |
| `getting-started.html` | Onboarding |

Built as static HTML/CSS/JS. Open `index.html` or view the hosted site via GitHub Pages. The left sidebar is shared verbatim across every page (see `app.js` for the scroll-spy, collapsible nav groups, and cross-page search).
